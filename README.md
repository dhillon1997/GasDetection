# GasDetection
**Hardware Components Required** <br/>
* Arduino UNO
* Breadboard
* MQ-5 Gas Sensor
* GSM module
* 16×2 LCD module.
* Potentiometer
* Male/Female Jumper Wires and Normal Wires
* 120 Ω Resistors <br/><br/>
**Software Requirements** <br/>
This project uses Arduino IDE as its primary software.
